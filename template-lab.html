Live LED Customizer Template Lab

How to run the experiment

  1. Load a template .syx you saved from Novation Components.
  2. Click Probe: rainbow pads (writes distinct values to each pad's value bytes) and Export.
  3. Send the exported file to the SL MkIII (Components, or a SysEx utility like SysEx Librarian / MIDI-OX).
  4. Select that template on the unit and look at the pads:
    • If colours change → we found a colour byte; tell me which pads changed and I'll build a proper colour editor around it.
    • If nothing changes → confirms templates can't carry LED colours; the live InControl tool is the only route.

The value bytes probed (offset 15/22/29 for pads) are, on the evidence so far, control value-ranges (max = 127) — not colour. This experiment is here to prove or disprove that on real hardware.
